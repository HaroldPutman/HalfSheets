# Half Sheets

An application to cut Letter size sheet music pages in half and save them in a new PDF with twice
as many pages to be displayed in an app like MobileSheets in Landscape mode.

## Quick Start

./Scripts/open-app.sh

1. Drag a PDF into the app/
2. Move the cutline to a reasonable place on each page
3. Click Export PDF... to save the results
